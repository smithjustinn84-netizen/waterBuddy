# Water Buddy Tech Stack

- **Language**: Kotlin 2.x
- **UI**: Compose Multiplatform (Material 3)
- **DI**: Metro (Dependency Injection)
- **DB**: Room KMP (Persistence)
- **Async**: Coroutines & Flow
- **Architecture**: MVVM + UFD + Navigator Pattern
- **Testing**: Kotest, Turbine, Mockkery
- **CI/CD**: Kover (Coverage), Spotless (Linting)
- **Navigation**: Type-safe Jetpack Navigation (Compose)
